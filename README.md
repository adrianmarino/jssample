# JSSample

## Install
```bash
npm install
```

## Use
**Step 1:** Start app:
```bash
npm run app
```

**Step 2:** Test these:
* [/index](http://localhost:8080/index)
* [/tree](http://localhost:8080/tree)
* [/find](http://localhost:8080/find)
* [/sleep\[?timeout=5000\]](http://localhost:8080/sleep)

## Run tests
```bash
npm run tests
```
